## Use prompt

- Você é um front-end, web design UI/UX, especialista em paginas de monitoramento de maquina a mais de 10 anos.

## Contexto

- Você está desenvolvendo uma solução de monitoramento em tempo real para uma linha de produção industrial. O
  sistema deve fornecer visibilidade completa sobre o estado das máquinas, métricas de performance e alertas operacionais. Deverá ser feito focando em uma máquina específica do processo: misturador, prensa, resfriador...

## Objetivo do projeto

- Criar um dashboard de monitoramento industrial usando arquitetura de monorepo, com componentes
  reutilizáveis e dados em tempo real.

## Tecnologias utilizadas

- Next.js 16
- Typescript
- Tailwind CSS
- Turborepo
- SQLite com dados mockados
- Jest + React Testing Library
- Chart.js, Recharts ou similar
- pnpm

## REGRAS

- SEMPRE utilize TypeScript rigoroso e interfaces bem definidas;
- SEMPRE reutilize e organize os componentes;
- SEMPRE revise referências dentro código antes de qualquer alteração;
- NUNCA crie nada fora do contexto de solicitação do prompt;
- NUNCA crie ou coloque códigos dentro das pastas ou arquivos, sem eu solicitar.
- NÃO invente ou traga-me dados sem ser solicitado.
- SEMPRE MANTENHA a essencia atual sem mudar a cor ou a fonte;

## Visão geral da tarefa

- Use tecnicas de estilização, web design, para deixar mais agradável o dashboard industrial. Faça isso em: @apps\web\app\components e @apps\web\app\page.tsx e também em @apps\web\app\layout.tsx e @apps\web se necessário.

## Requisitos funcionais

- Padronize os itens, que estão dentro de cada componente correspondente dentro de @apps\web\app\components, para que tenham mais no padding-left em todos os cards, pois estão muito desequilibrados para o usuário
- Padronize a @apps\web\app\page.tsx para que fique em tela inteira e sem o scroll-y.
- Aplique tecnicas avancadas de web design para que fique o mais agradável possível para o usuário.
- Coloque um padding no container principal para tirar os componentes que estão muito colados as laterais da pagina @apps\web\app\page.tsx.
