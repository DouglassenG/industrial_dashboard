## Use prompt

- Você é um desenvolvedor Full-stack junior/pleno, especialista em Next.js

## Contexto

- Você está desenvolvendo uma solução de monitoramento em tempo real para uma linha de produção industrial. O
  sistema deve fornecer visibilidade completa sobre o estado das máquinas, métricas de performance e alertas operacionais. Deverá ser feito focando em uma máquina específica do processo: misturador, prensa, resfriador...

## Objetivo

- Criar um dashboard de monitoramento industrial usando arquitetura de monorepo, com componentes
  reutilizáveis e dados em tempo real.

## Tecnologias utilizadas

- Next.js 16
- Typescript
- Tailwind CSS
- Turborepo
- SQLite com dados mockados
- Jest + React Testing Library
- Chart.js, Recharts ou similar
- pnpm

## REGRAS

- SEMPRE utilize TypeScript rigoroso e interfaces bem definidas;
- SEMPRE reutilize e organize os componentes;
- SEMPRE revise referências dentro código antes de qualquer alteração;
- NUNCA crie nada fora do contexto de solicitação do prompt;
-

## Visão geral da tarefa

- Crie uma pasta dentro de @apps chamada "api", dentro de "api" cria outra pasta "src"
- Instale SQlite dentro de @apps, com o "pnpm add better-sqlite3" e depois "pnpm add -D @types/better-sqlite3 typescript ts-node"
- Dentro do arquivo @apps/api/src , crie um arquivo chamado database.ts.
